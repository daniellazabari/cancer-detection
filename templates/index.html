Breast Cancer Prediction Service

Welcome to my Breast Cancer Prediction Service. This web-based service uses machine learning models trained on the Wisconsin Breast Cancer (Diagnostic) dataset to predict whether a breast mass is malignant or benign, based on user-provided data.

This project is a demonstration of my proficiency in building and deploying machine learning models. It involves various stages, such as data preprocessing, model building, model evaluation, and deploying the model in a web application using Flask and Docker.
